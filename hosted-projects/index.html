Wait, its not just Harkenburg?

The Harkenburg Build Server hosts not only the Harkenburg Minecraft Project, but is also home to many other creative Minecraft projects. These projects range from real-life city replicas to imaginative fictional cities. Each project is unique in itself, and they are all different from one another.

All these projects call the Harkenburg Build Server home. Each project has a respective Project Manager. These Project Managers run and manage their own projects. Each project that shares the server is separate and independent from the Harkenburg Project. The only commonality between them all is sharing the same build server.
